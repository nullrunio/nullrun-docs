What you get out of the box

⏱
Budget gate
Set a per-workflow cap in cents. The SDK asks the gateway "any budget left?" before every @protect call — no round-trip cost when the answer is "yes".
🛑
Circuit breaker
Auto-halt on runaway loops, retry storms, and budget blowups. Defaults to PERMISSIVE on outage for non-sensitive tools — sensitive tools always fail closed.
🛰
Real-time kill / pause
A WebSocket control plane pushes KILL and PAUSE to every connected SDK in milliseconds. WorkflowKilledInterrupt is a BaseException so it reaches the top of the agent loop, not a swallowed except Exception.
🔐
Sensitive-tool guard
Mark any function @sensitive and it must pass the gateway's pre-execution check before the body runs. Stripe charges, email sends, file deletes — all gate-able.
🔁
Auto-instrumentation
nullrun.init() patches OpenAI, Anthropic, LangGraph, OpenAI Agents, Mistral, Gemini, Cohere, Bedrock, LlamaIndex, CrewAI, and AutoGen — cost tracking without @protect.
📜
Decision history
Every gate decision (allow / block / flag / require-approval) is logged with the policy version, the projected cost, and the call context — auditable for SOC 2 and HIPAA workflows.
